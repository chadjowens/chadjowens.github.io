US Playing Cards

Proof of Concept – Poker Night Game Application (iPhone / iPad)

The Assignment

For US Playing cards I was asked to design a fully functional proof of concept game that would allow players to experience several styles of poker (i.e. Texas Hold ‘em, Omaha Eight, Seven Card Stud, etc) via turn-based game play with an iPad or iPhone.

The game would allow for a player to plan a game, invite multiple friends to participate and even host multiple game sessions simultaneously. It also incorporated all the components of a real table-based game including table talk which required the design of an integrated messaging platform.  

Additionally, the game manager could leverage email or Facebook as a way to invite players to participate in a game. The game was designed and developed with an agile methodology.

iPad screens samples with visual design applied.
