Sam’s Club

Photo Print Native Mobile Application

The Assignment

While working at Rockfish Interactive (now part of VMLY&R), I worked on a number of native mobile applications (phone and tablet) . One of my favorite projects was to design an experience that would allow a user to select photos from their mobile device and send them to a physical Sam's Club print service.

This project was deceivingly complex as I had to design solutions that allowed the app to also integrate with the full desktop browser ".com" experience as well as allow a full e-commerce purchase path. With a signed in state, the app would also keep track of order history for quick print reorders and allow the user to select which store was closest to their home (or current location) to pick up prints when ready.
‍

iPhone screen flows with visual design and annotations.

As this application was very utilitarian, the visual design leveraged the current Sam's Club band color palette. However, decisions made contributed to an overall digital style guide effort that was happening concurrently.
‍

iPhone concept screen flows and interaction design.

Most of the critical functionality of the application was thought through and documented in the App Map / Screen Flow and Wireframe phase. Developers were critical to the early concepting and ideation to insure feasibility.

Detailed interaction notes were included as part of the interaction design documentation for developers in addition to a low fidelity clickable prototype for key screen flows.
